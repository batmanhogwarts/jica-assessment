# J-Industries Cognitive Assessment (JICA)

A pre-employment cognitive assessment platform built for AP Psychology.

**Designed by Jonathan Shiell**

## Features

- 35-question cognitive assessment across 5 categories
- Analytical Reasoning, Pattern Recognition, Working Memory, Adaptive Thinking, Processing Efficiency
- Real-time scoring with radar chart visualization
- Detailed question-by-question breakdown
- Comprehensive methodology documentation
- Instructor access mode

## Setup

```bash
npm install
npm start
```

## Tech Stack

- React 18
- Tailwind CSS
- Recharts
